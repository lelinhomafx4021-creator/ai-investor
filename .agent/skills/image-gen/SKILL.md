# 图片生成技能

当用户要求生成图片时，请遵循以下指南：

## 使用场景

1. **UI 设计图** - 网页、App 界面设计稿
2. **图标素材** - Logo、按钮图标、功能图标
3. **插图配图** - 文章配图、产品展示图
4. **原型图** - 低保真/高保真原型设计

## 生成规范

### 1. UI 设计图

- 使用现代化设计风格（毛玻璃、渐变、阴影）
- 配色要和谐，避免大红大绿
- 布局要清晰，重点突出
- 不要生成设备边框（除非用户要求）

### 2. 图标素材

- 风格统一（线性/填充/扁平）
- 尺寸合适
- 颜色简洁

### 3. 配图插图

- 风格与项目匹配
- 避免违规内容
- 画面清晰

## 生成流程

1. 先确认用户需求（尺寸、风格、用途）
2. 使用 generate_image 工具生成
3. 询问用户反馈，必要时迭代优化

## 提示词技巧

- 描述要具体，不要太抽象
- 指定风格（现代/极简/可爱/科技感）
- 指定配色（暗色/亮色/特定颜色）
- 指定构图（居中/对称/留白）
